# 🚀 Quick Setup Guide

## To run this ChatGPT Clone:

1. **Get OpenRouter API Key** (free at openrouter.ai)
2. **Update the API key** in `backend/PRODUCTION_SERVER.py` line 25
3. **Start backend:** `python backend/PRODUCTION_SERVER.py` 
4. **Start frontend:** `npm run dev` in frontend folder
5. **Open:** http://localhost:3001 (or 3002)
