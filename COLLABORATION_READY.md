## 🛠️ **Quick Start for Ankit Kumar**

### Backend Setup
```bash

pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

- **API Documentation**: http://localhost:8000/docs
